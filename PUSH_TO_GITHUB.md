# Push to GitHub - Quick Reference

## Repository Name Suggestions
1. **bloom-filter-cpp** (recommended) - Clear and descriptive
2. **high-performance-bloom-filter** - Matches implementation plan
3. **bloomfilter** - Simple and clean

## Steps

1. **Create repository on GitHub:**
   - Visit: https://github.com/new
   - Choose a name from above
   - Don't initialize with README/gitignore/license
   - Create repository

2. **Add remote and push:**
   ```bash
   # Replace YOUR_USERNAME and YOUR_REPO_NAME
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

   Or with SSH:
   ```bash
   git remote add origin git@github.com:YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

3. **Done!** Your code is on GitHub.
